To install first upload this folder to your website. Then visit mysite.com/kidney-cms in a browser and fill out the information. If you want more options edit kidney-cms/config.php.
How it works:
Once you've installed the app, all you have to do to post to your blog is to email the address you specified in installation with the subject of the email being the post title and the message being the post content. Its that easy!
